2. Design and Development

The website was structured using HTML and styled using CSS. A clean and simple layout was chosen with clear navigation link to connect all 5 pages, consistent soft and professional colors, readable fonts, and spacing to improve user experience. The homepage hass my introduction, button links to my contact and project pages, and my skills. The contact page is created using "form" tag with my details and a direct mailto form.

3. Challenges

One challenge was creating a responsive layout using only CSS for both mobile and deskotop which was solved by using flexible containers and relative units. Maintaining consistent styling across all the pages was also a challenge. W3C HTMLand CSS validator helped me ensure my codes were upto web standards.

4. Testing

Each HTML file in the project folder was validated individually using the W3C HTML Validator through the file upload method. The CSS file was also validated using the W3C CSS Validator to ensure compliance with web standards.
